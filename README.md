# Ecommerce Project

This is a full stack ecommerce web application.

## Tech Stack

Frontend
- Next.js
- React
- Tailwind CSS

Backend (Coming Soon)
- Django REST Framework
- PostgreSQL
- JWT Authentication

Payment
- Razorpay Integration (planned)

## Features

- Product listing
- Add to cart
- Checkout system
- Secure authentication
- Online payment
